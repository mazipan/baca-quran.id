## Local Setup

### Requirement

- Node v20 (use [nvm](https://github.com/nvm-sh/nvm) or other version manager)
- pnpm v9: https://pnpm.io/installation

### Install Dependencies

```bash
pnpm i
```

### Run in your local

```bash
pnpm run dev
```

### Deployment

```bash
pnpm run build
```

Copy `build` folder to your hosting.
